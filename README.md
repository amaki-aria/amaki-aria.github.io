### front-matter

具体的书名应写在 `魔女之吻乃百合之味1：梦与希望的女神/_index.md`

```
---
title: 魔女之吻乃百合之味1：梦与希望的女神
author: amaki-aria
date: 2019-06-17
weight: 1
---
这里可以写简介
```

第一卷具体的卷标题应写在 `魔女之吻乃百合之味1：梦与希望的女神/第一卷/_index.md`

```
---
title: 第一卷 梦幻物语
author: amaki-aria
date: 2019-08-03
weight: 1
---
这里可以写简介
```

第一章具体的标题应写在 `魔女之吻乃百合之味1：梦与希望的女神/第一卷/第一章/_index.md`

```
---
title: 第一章 一切物语开始的地方

```

### 排版规范

#### 尽量使用 Markdown 语法

#### 不应该重复标题

错误示范：

```markdown
+++
title = "第十章 苏醒的魔导书"
weight = 10
+++
第一卷：梦幻物语 第10章

　　第十章 苏醒的魔导书

　　走在走廊上的时候，突然空中打开了一扇大门，“姐姐，刚刚收到了学生会的通知，好像是说在图书馆二层那边，有魔导书飞出来了。”走出来的，正是已经变身成了一只小猫娘的优佳，她脸上的表情一脸凝重，看起来似乎是发生了一些很严重的事情。
```

正确示范：

```markdown
+++
title = "第十章 苏醒的魔导书"
weight = 10
+++
　　走在走廊上的时候，突然空中打开了一扇大门，“姐姐，刚刚收到了学生会的通知，好像是说在图书馆二层那边，有魔导书飞出来了。”走出来的，正是已经变身成了一只小猫娘的优佳，她脸上的表情一脸凝重，看起来似乎是发生了一些很严重的事情。
```
